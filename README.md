# مساعد الاجتماعات الذكي — Smart Meeting Scribe

نظام آلي لتفريغ الاجتماعات وتلخيصها واستخراج بنود العمل (Action Items) وإرسال المحضر للمشاركين، مع دعم للغة العربية.

مشروع تخرج — قسم تقنية المعلومات (IT).

---

## 🎯 نبذة سريعة

يستقبل النظام تسجيلاً صوتياً لاجتماع، ثم:
1. يحوّل الصوت إلى نص (Speech-to-Text) عبر Whisper.
2. يلخّص المحتوى ويستخرج القرارات وبنود العمل عبر NLP.
3. يخزّن النتيجة ويولّد محضراً منظّماً (PDF).
4. يرسل المحضر والمهام للمشاركين بالبريد الإلكتروني.

---

## 👥 القطاعات الثلاثة وهيكل المجلدات

```
smart-meeting-scribe/
├── frontend/      # القطاع الأول (المسؤول): واجهة المستخدم + التكامل النهائي
├── ai-engine/     # القطاع الثاني: Whisper + NLP (تفريغ، تلخيص، استخراج مهام)
├── backend/       # القطاع الثالث: الخادم + قاعدة البيانات + البريد
├── docs/          # التوثيق وعقود الربط (api-contract.md)
└── README.md
```

| القطاع | المسؤولية | المجلد |
|--------|-----------|--------|
| **الأول (Lead)** | الواجهة + توحيد عقود الربط + تجميع النظام على جهاز واحد | `frontend/` |
| الثاني | تحويل الكلام إلى نص + التلخيص + استخراج بنود العمل | `ai-engine/` |
| الثالث | الخادم الخلفي + قاعدة البيانات + توليد المحضر + البريد | `backend/` |

> القطاع الأول هو المنسّق: يملك القرار النهائي في عقود الربط ودمج الأجزاء.

## 🚀 خطوات البدء (Getting Started)

### 1. استنساخ المستودع
```bash
git clone <repo-url>
cd smart-meeting-scribe
```

### 2. أدوات أساسية للجميع
- Git
- Python 3.10+
- Node.js (للواجهة)
- ffmpeg  ← إلزامي لـ Whisper

### 3. تثبيت كل قطاع

**قطاع الذكاء الاصطناعي:**
```bash
cd ai-engine
python -m venv venv
source venv/bin/activate      # على ويندوز: venv\Scripts\activate
pip install -r requirements.txt
```

**قطاع الخادم:**
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

**قطاع الواجهة:**
```bash
cd frontend
npm install
npm start
```

## 🌿 قواعد العمل على Git

- كل قطاع يعمل على فرع خاص: `feature/frontend`، `feature/ai`، `feature/backend`.
- الدمج في الفرع الرئيسي (`main`) يتم عبر Pull Request يراجعه القطاع الأول.
- لا يُرفع أي ملف كبير (نماذج Whisper، ملفات صوت، `venv/`، `node_modules/`) — تُضاف إلى `.gitignore`.

### مثال `.gitignore`
```
venv/
node_modules/
__pycache__/
*.mp3
*.wav
*.pt
.env
```
